# דוד - מערכת שוע"ל אזרחי 🛡️

בוט טלגרם שמרליי לצ'אט/ערוץ:
- התרעות בזמן אמת מ-API הציבורי של פיקוד העורף
- פוסטים מערוץ טלגרם ציבורי לבחירתכם (ברירת מחדל: `CaptainLoui`)
- מפה חיה של ישראל שמדגישה אזורים עם התרעה פעילה, נגישה בדפדפן

⚠️ **הבהרה**: זהו כלי קהילתי לא רשמי שמרליי מידע ציבורי. לכל התרעה יש תיוג "מידע לא רשמי" עם הפניה למקור הרשמי. אין להסתמך עליו כמקור בלעדי להנחיות חירום.

## התקנה

```bash
npm install
cp .env.example .env
```

מלאו ב-`.env`:
- `BOT_TOKEN` - טוקן מ-[@BotFather](https://t.me/BotFather)
- `CHAT_ID` - ה-ID של הצ'אט/ערוץ/קבוצה שהבוט ישלח אליו (ראו הוראות מפורטות למטה)
- `TELEGRAM_SOURCE_CHANNEL` - שם המשתמש (בלי `@`) של הערוץ הציבורי שממנו הבוט יריליי פוסטים

## פתיחת ערוץ וחיבור הבוט

**1. יצירת הערוץ**
- באפליקציית טלגרם: תפריט (☰) → *ערוץ חדש*
- שם: `דוד - מערכת שוע"ל אזרחי 🛡️`
- מומלץ להוסיף בתיאור שזהו כלי קהילתי לא רשמי
- ניתן לבחור ציבורי (עם `@username`) או פרטי

**2. הוספת הבוט כמנהל**
- בהגדרות הערוץ/קבוצה → *מנהלים* → *הוספת מנהל*
- מחפשים את שם המשתמש של הבוט (שקיבלתם מ-@BotFather)
- מפעילים לפחות את ההרשאה *פרסום הודעות*

**3. קבלת ה-CHAT_ID**
- **ערוץ/קבוצה ציבוריים** (יש להם `@username`) - הכי פשוט, שימו ב-`.env`:
  ```
  CHAT_ID=@your_channel_username
  ```
- **ערוץ/קבוצה פרטיים** - צריך את ה-ID המספרי:
  1. פרסמו הודעה כלשהי בערוץ/בקבוצה (אחרי שהבוט כבר חבר בה)
  2. גשו בדפדפן ל-`https://api.telegram.org/bot<הטוקן שלכם>/getUpdates`
  3. חפשו בתשובה JSON את `"chat":{"id":-100...}` - זה המספר (בדרך כלל יתחיל ב-`-100` לערוצים/קבוצות)
  4. שימו אותו ב-`CHAT_ID` ב-`.env`

## הרצה

```bash
npm start
```

לאחר ההרצה, המפה החיה זמינה בדפדפן בכתובת: `http://localhost:3000` (כל עוד התהליך רץ).

## מבנה הפרויקט

```
src/
  config.js                     # הגדרות, מיתוג, ערוץ המקור, פורט
  formatters.js                 # תבניות ההודעות (סגנון "התרעת חירום")
  bot.js                        # מופע ה-Telegraf, פקודות /start ו-/status
  server.js                     # שרת ה-Express שמגיש את המפה החיה
  index.js                      # נקודת הכניסה, לולאות ה-polling
  data/
    areaCoordinates.js          # מיפוי ערים/אזורים לקואורדינטות למפה
  services/
    orefService.js              # שליפת התרעות מ-API של פיקוד העורף
    telegramChannelService.js   # שליפת פוסטים מערוץ הטלגרם המקור
public/
  index.html                    # דף המפה החיה (Leaflet)
```

## התאמה אישית

- ערוץ מקור אחר: `TELEGRAM_SOURCE_CHANNEL` בקובץ `.env`
- תדירות בדיקה: `OREF_POLL_INTERVAL_MS` ו-`NEWS_POLL_INTERVAL_MS` בקובץ `.env`
- פורט המפה: `PORT` בקובץ `.env` (ברירת מחדל 3000)
- עיצוב ההודעות: `src/formatters.js`
- אזורים נוספים על המפה: `src/data/areaCoordinates.js`

## הערה על המפה החיה

פיקוד העורף מדווח על כ-1000+ אזורי התרעה מדויקים (ברמת שכונה). המפה מכילה מיפוי ידני של כ-60 ערים/אזורים מרכזיים בלבד - התרעה באזור שלא במיפוי עדיין מגיעה לבוט הטלגרם, אך תופיע כרשימת טקסט מתחת למפה במקום כסמן.

## הערה על ערוץ המקור

הבוט שולף פוסטים מהעמוד הציבורי (`t.me/s/<שם הערוץ>`) של הערוץ שהגדרתם - זו הדרך היחידה לקרוא תוכן מערוץ שהבוט אינו מנהל בו, כי ה-Bot API הרשמי לא מאפשר קריאת פוסטים מערוצים זרים. חשוב לוודא שהערוץ שבחרתם ציבורי ואמין, כי הבוט יריליי כל מה שמתפרסם בו כמות שהוא.
